Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page's structure in memory as a logical tree of nodes and objects. It allows developers to manipulate the page's content, style, and structure using scripting languages like JavaScript

How do you select an element from DOM?

To select an element from the DOM:

1. Use document.getElementById('id') to select by ID.

2. Use document.querySelector('selector') to select the first match of any CSS selector.

3. Use document.getElementsByClassName('class') or document.getElementsByTagName('tag') for class andtag selections

What is the event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners on individual child elements. The event bubbles up from the target child to the parent, where the listener can handle events from all child elements.

Why it's useful: Efficiency: Reduces memory usage by attaching fewer event listeners. Dynamic Elements: Handles events for dynamically added child elements without needing to reassign listeners. Simpler Management: Centralizes event handling in one place, making the code cleaner and easier to maintain.

How do you manipulate an element`s attributes and styles using DOM?

You can manipulate an element's attributes and styles using the DOM in the following ways:

1. Attributes: Use element.setAttribute('attr', 'value') to set an attribute or element.getAttribute('attr') to get it.

2. Styles: Access or modify styles via element.style.property = 'value' (e.g., element.style.color = 'red';).

3. Classes: Use element.classList.add('class'), element.classList.remove('class'), or element.classList.toggle('class') to manage CSS classes
